# 📋 JSON 字段规整规则说明

## 一、资格条件 (资格条件)

### 1. 学历要求
**结构**: `{"条件": "", "排名": [], "学历": []}`

#### 规则：
- **条件判断**:
  - 包含 "任一"、"或" → `"条件": "或"`
  - 包含 "且"、"同时" → `"条件": "且"`
  - 其他 → `"条件": ""`

- **排名提取** (放入 `排名` 数组):
  - 包含 "排行榜"、"QS"、"泰晤士" 等关键词的完整句子
  - 包含 "985"、"211"、"双一流" 的完整句子
  
- **学历提取** (放入 `学历` 数组):
  - 其他学历相关描述
  - 按句号、分号、逗号（用于分隔完整条件）分割

**示例**:
```
原文: "原则上应满足以下条件任一：①全日制硕士研究生及以上学历【国（境）外高校进入"四大"世界大学排行榜前300名】，②全日制本科为原985、211院校。"

规整后:
{
  "条件": "或",
  "排名": ["全日制硕士研究生及以上学历【国（境）外高校进入\"四大\"世界大学排行榜前300名]"],
  "学历": ["全日制本科为原985、211院校。"]
}
```

---

### 2. 专业要求
**结构**: `{"条件": "", "专业": [], "经历": []}`

#### 规则：
- **条件判断**:
  - 包含 "或" → `"条件": "或"`
  - 包含 "且"、"同时" → `"条件": "且"`
  
- **专业提取** (放入 `专业` 数组):
  - 按 "，" 分割专业类别
  - 以 "类"、"专业" 结尾的词组
  
- **经历提取** (放入 `经历` 数组):
  - 包含 "具备"、"工作经历"、"年" 的完整句子

**示例**:
```
原文: "电气类，自动化类，控制科学与工程类，能源动力类，相关专业。或具备1年发电技能工作经历。"

规整后:
{
  "条件": "或",
  "专业": ["电气类", "自动化类", "控制科学与工程类", "能源动力类", "相关专业"],
  "经历": ["或具备1年发电技能工作经历。"]
}
```

---

### 3. 年龄要求
**结构**: `""`  (字符串)

#### 规则：
- 提取数字 + 比较符号
  - "不超过XX周岁" → `"≤XX"`
  - "XX周岁以下" → `"<XX"`
  - "XX周岁及以上" → `"≥XX"`
  - "XX-YY周岁" → `"XX-YY"`

**示例**:
```
原文: "一般不超过40周岁。"
规整后: "≤40"
```

---

### 4. 绩效要求
**结构**: `{"条件": "", "系统内": "", "系统外": ""}`

#### 规则：
- **条件判断**:
  - 同时包含 "系统内" 和 "系统外" → `"条件": "与"`
  
- **系统内/系统外提取**:
  - 按 "系统内" 和 "系统外" 分割文本
  - 提取对应的要求描述

**示例**:
```
原文: "系统内，近3年个人年度绩效考核结果均为称职（B级）及以上。系统外，近3年个人年度绩效考核结果均不得为不及格。"

规整后:
{
  "条件": "与",
  "系统内": "近3年个人年度绩效考核结果均为称职（B级）及以上。",
  "系统外": "近3年个人年度绩效考核结果均不得为不及格。"
}
```

---

### 5. 职称要求
**结构**: `[]`  (数组)

#### 规则：
- 识别职称等级关键词:
  - "正高级" → 加入数组
  - "副高级" → 加入数组
  - "中级" → 加入数组
  - "初级" → 加入数组
  - "高级及以上" → 加入 ["正高级", "副高级"]
  - "中级及以上" → 加入 ["正高级", "副高级", "中级"]

**示例**:
```
原文: "具有中级及以上职称。"
规整后: ["正高级", "副高级", "中级"]
```

---

### 6. 工作经历
**结构**: `{"条件": "", "南方电网公司系统内应聘人员": "", "南方电网公司系统外应聘人员": ""}`

#### 规则：
- **条件判断**:
  - 包含 "（1）" 和 "（2）" → `"条件": "或"`
  
- **内容提取**:
  - 按 "（1）" 和 "（2）" 或 "系统内" 和 "系统外" 分割
  - 提取对应的要求描述，去除标号

**示例**:
```
原文: "（1）南方电网公司系统内应聘人员：...；（2）南方电网公司系统外应聘人员：..."

规整后:
{
  "条件": "或",
  "南方电网公司系统内应聘人员": "...",
  "南方电网公司系统外应聘人员": "..."
}
```

## 二、岗位任职条件 (岗位任职条件)

### 1. 工作经验
**结构**: `""`  (字符串)

#### 规则：
- 提取年限数字
  - "X年及以上" → `"≥X"`
  - "X年以上" → `">X"`
  - "X-Y年" → `"X-Y"`

**示例**:
```
原文: "3年及以上工作经验。"
规整后: "≥3"
```

---

### 2. 能力要求
**结构**: `[]`  (字符串数组)

#### 规则：
- 保持原文，按条目分割
- 通常以 "（1）"、"（2）" 等标号分割

**示例**:
```
能力要求: [
  "（1）具备业务需求分析能力；",
  "（2）具备创新思维；"
]
```

---

### 3. 持证要求
**结构**: `""`  (字符串)

#### 规则：
- 直接复制原文内容

**示例**:
```
原文: "具备一级建造师职业资格，同时具有高级工程师或一级造价工程师优先。"
规整后: "具备一级建造师职业资格，同时具有高级工程师或一级造价工程师优先。"
```

---

## 三、关键词映射表

### 条件关键词
- **"或"**: "任一"、"或"、"可"
- **"且"**: "且"、"同时"、"并"、"均"

### 职称等级映射
- **正高级**: "正高级"、"研究员级"
- **副高级**: "副高级"、"高级"
- **中级**: "中级"
- **初级**: "初级"、"助理级"

### 比较符号映射
- **≥**: "及以上"、"以上（含）"、"不少于"
- **>**: "以上"、"超过"
- **≤**: "不超过"、"以下（含）"、"不多于"
- **<**: "以下"
