# AI简历初筛系统 - 部署包使用说明

## 📦 部署包内容

```
部署包/
├── AI简历初筛系统.exe    # 主程序（双击运行）
├── data/                  # 数据目录（自动创建，存放解析后的JSON文件）
├── output/                # 输出目录（自动创建，存放筛选结果）
└── logs/                  # 日志目录（自动创建，存放运行日志）
```

## 🚀 快速开始

### 1. 运行程序

双击 `AI简历初筛系统.exe` 启动服务。

**首次运行**：
- 程序会自动创建必要的目录（data、output、logs）
- 控制台窗口会显示服务启动信息
- 浏览器会自动打开 `http://127.0.0.1:8000`

### 2. 使用界面

1. **上传文件**
   - 选择简历文件：`（现RPA小工具流程）简历导入多行表.xlsx`
   - 选择岗位文件：`条件要求较简单的部分岗位岗位要求-模拟数据.xlsx`
   - 点击"🚀 开始处理"按钮

2. **等待处理**
   - 系统会自动解析Excel文件
   - 执行AI筛选（使用阿里云DashScope API）
   - 处理完成后显示结果

3. **查看结果**
   - 在网页界面查看筛选结果
   - 结果会自动保存到 `output/简历初筛结果.json`
   - 解析后的JSON文件保存在 `data/` 目录

## ⚙️ 配置说明

### API密钥配置

如果需要修改LLM API配置，请编辑 `config.py` 文件（如果存在）或联系管理员。

**注意**：打包后的exe文件中已包含配置，如需修改需要重新打包。

### 端口配置

默认端口：`8000`

如需修改端口，需要修改源代码后重新打包。

## 📁 目录说明

### data/ 目录
- 存放解析后的简历JSON文件
- 存放解析后的岗位JSON文件
- 包含系统数据文件（专业库.json、院校库.json）

### output/ 目录
- 存放筛选结果JSON文件
- 文件名：`简历初筛结果.json`

### logs/ 目录
- 存放运行日志
- 日志文件：`resume_filter.log`

## 🔧 常见问题

### 1. 程序无法启动

**可能原因**：
- Windows Defender或其他杀毒软件拦截
- 缺少必要的系统库（Visual C++ Redistributable）

**解决方法**：
- 将exe文件添加到杀毒软件白名单
- 安装 [Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)

### 2. 浏览器未自动打开

**解决方法**：
- 手动打开浏览器
- 访问：`http://127.0.0.1:8000`

### 3. 上传文件后处理失败

**检查项**：
- Excel文件格式是否正确
- 文件是否损坏
- 查看控制台窗口的错误信息
- 查看 `logs/resume_filter.log` 日志文件

### 4. API调用失败

**可能原因**：
- 网络连接问题
- API密钥无效或过期
- API配额用尽

**解决方法**：
- 检查网络连接
- 联系管理员检查API配置
- 查看日志文件了解详细错误

### 5. 端口被占用

**错误信息**：`Address already in use`

**解决方法**：
- 关闭占用8000端口的其他程序
- 或修改源代码中的端口号后重新打包

## 📊 系统要求

- **操作系统**：Windows 10/11 (64位)
- **内存**：建议 4GB 以上
- **磁盘空间**：至少 500MB 可用空间
- **网络**：需要互联网连接（用于调用LLM API）

## 🔒 安全说明

1. **API密钥**：已内置在exe文件中，请妥善保管
2. **数据隐私**：所有处理都在本地进行，不会上传到第三方服务器（除了LLM API调用）
3. **文件安全**：上传的文件仅用于处理，不会泄露
